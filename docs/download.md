# Download Kelivo

Get Kelivo on your device - available for Android, iOS, and HarmonyOS.

## üì± iOS

### App Store (Recommended)
Stable releases for iPhone and iPad.

[![Download on the App Store](https://developer.apple.com/assets/elements/badges/download-on-the-app-store.svg)](https://apps.apple.com/us/app/kelivo/id6752122930)

Early access to new features and updates.

**Requirements**: TestFlight app installed, iOS 14.0+

**Installation**:
1. Install TestFlight from App Store if you don't have it
2. Open the beta link on your iOS device
3. Tap "Accept" to join
4. Tap "Install" in TestFlight

**Note**: Beta builds may have bugs. Report issues on GitHub.

## ü§ñ Android

### APK Download
Direct download from GitHub Releases.

**Latest Release**: https://github.com/Chevey339/kelivo/releases/latest

**Requirements**: Android 8.0 (API 26) or later

**Installation Steps**:

1. **Download the APK**
   - Visit the releases page on your Android device
   - Tap the latest `.apk` file to download

2. **Enable Unknown Sources** (if needed)
   - Go to **Settings ‚Üí Security**
   - Enable **Install unknown apps** for your browser
   - Or enable **Unknown sources** (older Android)

3. **Install the APK**
   - Open your downloads or tap the notification
   - Tap the APK file
   - Tap **Install**
   - Tap **Open** when complete

**Troubleshooting**:
- **Can't install**: Check Android version (need 8.0+)
- **Security warning**: This is normal for apps outside Play Store
- **Installation blocked**: Enable unknown sources in settings

**Note**: Google Play Store release is planned for future.

## Update Instructions

### iOS (App Store)
Updates install automatically if auto-update is enabled.

**Manual update**:
1. Open **App Store**
2. Tap your **profile icon**
3. Scroll to **Available Updates**
4. Tap **Update** next to Kelivo

### iOS (TestFlight)
TestFlight notifies you of new beta builds.

**Manual check**:
1. Open **TestFlight**
2. Tap **Kelivo**
3. Tap **Install** if update available

### Android (APK)
No automatic updates. Check for new releases periodically.

**Update process**:
1. Visit [releases page](https://github.com/Chevey339/kelivo/releases/latest)
2. Download new APK
3. Install over existing app (data is preserved)

**Tip**: Enable notifications on the GitHub repo to get release alerts.

### HarmonyOS
Check the releases page for updates and install manually.

## Uninstallation

### iOS
1. **Long-press** the Kelivo icon on home screen
2. Tap **Remove App**
3. Select **Delete App**

Or via Settings ‚Üí General ‚Üí iPhone Storage ‚Üí Kelivo ‚Üí Delete App

### Android
1. **Settings ‚Üí Apps**
2. Find **Kelivo**
3. Tap **Uninstall**

Or long-press app icon ‚Üí App info ‚Üí Uninstall

### HarmonyOS
Follow standard HarmonyOS app removal process.

**Note**: Uninstalling removes all local data (chats, settings, API keys). Export data first if needed.

## Frequently Asked Questions

### Is Kelivo free?
Yes, the app is free and open-source. You'll need API keys from AI providers, which may have their own costs.

### Why isn't Kelivo on Google Play Store?
Play Store release is planned but not yet available. For now, download the APK directly.

### Is it safe to install APKs?
Yes, when from official sources. Always download from the official GitHub repository.

### How do I know if an update is available?
- **iOS App Store**: Check for updates in App Store
- **Android**: Watch the GitHub releases page or enable notifications
- **TestFlight**: Automatic notifications

### Can I install on multiple devices?
Yes! Install on all your devices and use export/import to transfer data between them.

### What if I have installation issues?
Check the [FAQ](/docs/getting-started/faq) or [open an issue](https://github.com/Chevey339/kelivo/issues) with:
- Device model
- OS version
- Error message or screenshot
- Steps you tried
